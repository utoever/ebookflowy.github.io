로컬 우선과 개인정보 ​

백업/복원 설정

ebookflowy는 원본 문서를 기본적으로 업로드하지 않는 로컬 우선 앱입니다. 개인·업무 문서를 외부에 올리지 않고도 읽고 관리하도록 설계했습니다.

로컬 저장 ​

읽던 위치, 북마크, 주석, 검색 색인은 모두 기기 안에 저장됩니다. 핵심 읽기·검색 기능은 네트워크 없이도 동작합니다.

외부 서비스는 사용자가 선택할 때만 ​

번역, AI, 동기화, 클라우드 TTS처럼 텍스트를 외부로 보내는 기능은 사용자의 명시적 동작 또는 opt-in이 있을 때만 사용됩니다.

  • 보낼 내용은 실행 전에 사용자에게 보여집니다.
  • 자동 전송 흐름을 만들지 않습니다.
  • 로컬 TTS는 읽을 텍스트만 엔진에 전달하고 원본을 서버로 올리지 않습니다.

클립보드 ​

클립보드는 자동으로 읽지 않습니다. 붙여넣기·사전·번역은 사용자가 직접 눌렀을 때만 동작합니다.

권한 범위 ​

  • Android: 사용자가 허용한 폴더/파일 범위 안에서만 접근(SAF)
  • iOS/iPadOS: 사용자가 선택한 문서/폴더 접근 권한을 앱 샌드박스 안에서 유지

백업과 복원 ​

북마크, 주석, 설정, 보관함 정보를 JSON으로 내보내고 복원할 수 있습니다.

  • 기본값은 메타데이터만 포함합니다.
  • 앱 내부 문서(공유로 가져온 파일, 만든 PDF 등) 포함은 사용자가 선택할 때만 추가됩니다.
  • 외부 원본 파일, 플랫폼 권한 토큰, 색인·썸네일 캐시는 백업에 담기지 않습니다.
  • 복원 후 외부 원본은 다시 선택하면 available 상태가 됩니다.

원본 보호 ​

주석과 하이라이트는 원본이 아닌 앱 내부에 저장됩니다. 원본을 바꾸는 작업은 별도 확인을 거치므로, 실수로 원본이 훼손되지 않습니다.
