# Markdown 읽기

*Markdown 렌더링 화면*

Markdown 문서는 WebView 기반 리더로 열려, 서식이 풍부하게 표시됩니다. 렌더링에 필요한 요소는 앱에 포함되어 있어 인터넷 없이도 동작합니다(이미지 제외).

## 지원하는 표현

- 제목, 목록, 표
- 코드 블록과 구문 강조
- Mermaid 다이어그램
- Markwhen 타임라인
- KaTeX 수식

Mermaid와 KaTeX 자원은 외부 CDN이 아니라 앱에 번들로 들어 있어, 오프라인에서도 다이어그램과 수식이 그려집니다.

## 안전한 렌더링

Markdown은 화면에 그리기 전에 정제(sanitize)된 HTML로 변환됩니다.

- 외부 스크립트, iframe, 인라인 이벤트 핸들러, 자동 화면 이동은 차단됩니다.
- 외부 링크는 자동으로 열리지 않고, 사용자가 직접 누른 경우에만 처리됩니다.
- 본문의 원격 이미지는 표시할 수 있습니다(이 경우 해당 이미지 주소로 네트워크 요청이 발생할 수 있습니다).

## 읽기 기능

Markdown 문서도 다른 포맷과 같은 읽기 기능을 사용합니다.

- 진행률 자동 저장과 복원
- 북마크와 다시 열기
- 현재 문서 검색과 보관함 본문 검색
- 세로 보기 자동 스크롤

## 범위

이 리더는 읽기 전용입니다. Markdown 편집과 Markdown 위의 주석은 제공하지 않습니다. 메모가 필요하면 북마크와 노트를 활용하세요.
