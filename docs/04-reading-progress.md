# 이어읽기와 독서중

*홈의 독서중 영역*

문서를 열면 읽던 위치가 자동으로 저장되어, 다음에 열 때 그 위치부터 이어 읽습니다. 따로 저장 버튼을 누를 필요가 없습니다.

## 자동 저장의 동작

- 페이지를 넘기거나 스크롤하면 현재 위치가 갱신됩니다.
- 앱을 닫았다 다시 켜도 마지막 위치가 복원됩니다.
- 진행률은 표지와 리더에 함께 표시됩니다.

## 독서중 큐

홈의 `독서중`은 실제 열람 활동을 기준으로 최근에 읽은 문서를 모아 둡니다. 바로 눌러 마지막 위치로 돌아갈 수 있어, "어제 보던 그 문서"를 찾아 헤맬 필요가 없습니다.

검색 결과로 잠깐 열어 본 문서까지 큐를 오염시키지 않도록, 가이드 문서나 일부 임시 열람은 독서중에 남기지 않습니다.

## Ebook 책장 순서는 그대로

`Ebook 책장`은 문서를 추가한 순서를 유지합니다. 같은 문서를 다시 읽어도 책장 순서는 바뀌지 않으므로, 늘 같은 자리에서 문서를 찾을 수 있습니다. "최근 읽은 순"으로 보고 싶다면 `독서중`을 사용하면 됩니다.

## 포맷별 위치 기준

| 포맷 | 위치 기준 |
|---|---|
| PDF | 페이지 번호 |
| TXT / Markdown | 본문 위치(offset) |
| EPUB | 챕터(spine)와 CFI |
| CBZ / CB7 | 이미지 순서 |
| HWP / HWPX | 페이지 |

각 포맷은 자기 방식대로 위치를 기억하므로, 글자 크기나 화면을 바꿔도 읽던 지점을 최대한 맞춰 복원합니다.

> 가이드 책장의 문서도 진행률과 북마크가 동작하지만, 독서중 큐에는 올라오지 않습니다.
