# 리더 설정

*환경설정 화면*

설정은 화면 오른쪽 위의 톱니바퀴 아이콘으로 들어갑니다. 읽기 환경을 취향과 눈 상태에 맞게 조절할 수 있습니다.

## 모양과 표시

- 화면 모드: 자동 / 라이트 / 다크 (3분할 토글)
- 글꼴, 줄간격, 테마
- 도구막대 위치

화면 모드 "자동"은 기기의 시스템 설정을 따릅니다.

## 포맷별 설정

TXT, Markdown, EPUB, CBZ는 포맷마다 별도(override) 설정을 가질 수 있습니다.

- 예: TXT는 큰 글씨와 넓은 줄간격, 만화는 다른 보기 방식
- 공통 설정을 기본으로 두고, 특정 포맷만 다르게 둘 수 있습니다.
- 자동 스크롤 속도도 포맷별로 따로 정할 수 있습니다.

## 접근성

- 고대비 모드로 글자와 배경 대비를 높입니다.
- 글자 크기 조절로 읽기 편한 크기를 맞춥니다.
- 주요 버튼과 행은 충분한 크기로 만들어 탭하기 쉽게 되어 있습니다.

## 앱 언어

앱 언어는 시스템 기본값을 따르며, 한국어·영어·베트남어·일본어로 직접 바꿀 수 있습니다(설정 > 기본).

- 시스템 언어가 지원되지 않으면 한국어로 표시됩니다.
- 직접 입력한 데이터(메모, 파일명 등)는 언어를 바꿔도 자동 번역되지 않습니다.

## 기타

- 기본 북마크 색 변경
- `가이드 책장` 표시 끄기/켜기
- 광고/프리미엄 항목에서 구매 복원 등 관리

> 설정 화면들은 값·상태를 명확히 보여 주도록 구성되어 있어, 바꾼 값이 바로 반영됩니다.
