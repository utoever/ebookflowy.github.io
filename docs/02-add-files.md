# 파일 추가와 권한

*추가 버튼과 문서 선택기*

홈 화면 오른쪽 아래의 `추가` 버튼으로 문서를 `Ebook 책장`에 담을 수 있습니다. 다른 앱에서 공유해서 가져올 수도 있습니다(자세한 내용은 `공유로 가져오기` 참고).

## 추가할 수 있는 포맷

| 종류 | 포맷 |
|---|---|
| 문서 | PDF, TXT, Markdown(`.md`/`.markdown`), HWP/HWPX |
| 책 | EPUB |
| 만화·이미지 | CBZ, CB7 |

`.md`와 `.markdown`은 Markdown 리더로, 일반 `.txt`는 TXT 리더로 자동 분기됩니다. TXT는 UTF-8, UTF-8 BOM, UTF-16, CP949/EUC-KR 인코딩을 자동으로 인식해 한글이 깨지지 않게 정규화합니다.

## 추가하는 방법

1. 홈에서 `추가` 버튼을 누릅니다.
2. 기기의 문서 선택기가 열리면 파일을 고릅니다.
3. 선택한 문서가 `Ebook 책장`에 추가됩니다.

## Android 권한 (SAF)

Android에서는 사용자가 선택한 파일 또는 폴더 범위 안에서만 문서를 읽습니다. 폴더를 한 번 허용하면 그 범위에 한해 접근 권한이 유지되므로, 같은 폴더의 문서는 다시 고르지 않아도 됩니다.

## iOS와 iPadOS 권한

iOS에서는 문서 선택기에서 허용한 파일을 엽니다. 보안 정책상 접근 권한이 만료되면 같은 파일을 다시 선택해야 할 수 있습니다. 이때는 "파일을 찾을 수 없음" 대신 다시 선택 안내가 표시되며, 안내에 따라 파일이나 폴더를 다시 고르면 복구됩니다.

## 원본은 안전하게

- 원본 문서는 기본적으로 앱이 직접 수정하지 않습니다.
- 북마크·하이라이트·주석은 원본이 아니라 앱 내부에 저장됩니다.
- 공유로 가져온 문서는 앱 내부 복사본으로 보관되어, 외부 원본이 사라져도 계속 열립니다.

> 폴더 전체를 한꺼번에 다루고 싶다면 `매직보관함`을, 다른 앱에서 보낸 자료를 담으려면 `공유로 가져오기`를 참고하세요.
