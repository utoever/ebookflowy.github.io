# PDF 만들기

*PDF 만들기 화면*

공유받은 자료나 직접 입력한 내용으로 새 PDF를 만들 수 있습니다. 흩어진 이미지나 웹 내용을 한 문서로 정리할 때 유용합니다.

## 만드는 방법

두 가지 경로가 있습니다.

- 외부에서 공유한 이미지·URL·텍스트로 PDF 생성
- 앱 안의 `PDF 만들기`에서 직접 입력해 생성

처리 방식은 다음과 같습니다.

- 이미지: 한 장 또는 여러 장을 PDF로 묶습니다.
- URL: 웹 내용을 PDF 문서로 변환합니다.
- 텍스트: 입력한 글을 PDF 문서로 만듭니다.

## 이미지 저장 형식

이미지를 가져올 때는 PDF 대신 CBZ(만화책 형식)로 저장할 수도 있습니다. 만화·웹툰처럼 페이지를 넘겨 보는 콘텐츠는 CBZ가 더 자연스럽습니다.

## 만든 다음

생성한 PDF는 앱 로컬 보관함으로 들어가 색인 대기 상태로 연결됩니다.

- 바로 열어 읽을 수 있습니다.
- 색인이 끝나면 보관함 본문 검색 대상이 됩니다.
- 다른 문서와 똑같이 북마크·진행률·공유를 사용할 수 있습니다.

> 새 파일을 만드는 작업이므로, 생성은 사용자가 직접 실행할 때만 진행됩니다.
