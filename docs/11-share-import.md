# 공유로 가져오기

*공유 가져오기 선택 화면*

다른 앱(브라우저, 파일, 사진 등)에서 ebookflowy로 `공유` 또는 `다음으로 열기`를 하면 내용을 보관함으로 가져올 수 있습니다. 파일을 일일이 저장한 뒤 추가하지 않아도 됩니다.

## 가져오는 방법

1. 다른 앱에서 공유 버튼을 누릅니다.
2. 공유 대상에서 ebookflowy를 고릅니다.
3. 항목에 따라 저장 방식이 정해지거나, 형식을 선택하는 화면이 나옵니다.

## 항목별 처리

- 지원 문서(PDF, EPUB 등): 보관함에 추가
- URL·링크 텍스트·일반 텍스트: 보관함 항목으로 처리
- 이미지: PDF 또는 CBZ 형식 중 선택해 저장
- 여러 이미지: 한 권의 PDF/CBZ로 묶기
- `.ebslide`: 슬라이드 책장의 슬라이드북으로 가져오기

## 열기 정책

가져온 항목은 홈·보관함(또는 슬라이드 책장)에 남습니다. 최신 기본 동작에서는 가져온 뒤 리더를 자동으로 열지 않으므로, 원하는 때에 직접 열어 보면 됩니다. 의도치 않게 읽기 흐름이 끊기지 않도록 한 설계입니다.

## 내부 복사본으로 안전하게

공유로 가져온 문서는 앱 내부에 관리 복사본으로 저장됩니다. 그래서 다음 상황에서도 계속 열 수 있습니다.

- 앱을 재시작한 뒤
- 외부 원본이 삭제된 뒤
- 외부 폴더 접근 권한이 만료된 뒤

반대로, 외부 원본을 직접 참조하는 문서(Android SAF/iOS 북마크)는 권한이 만료되면 다시 선택이 필요할 수 있습니다.

> 가져온 이미지로 새 PDF를 만드는 방법은 `PDF 만들기`도 함께 참고하세요.
