# 검색 사용하기

*검색 결과와 스니펫*

ebookflowy는 파일명 검색과 본문 검색을 구분해서 다룹니다. "지금 읽는 문서 안"과 "보관함 전체"를 따로 찾을 수 있습니다.

## 현재 문서 안에서 검색

리더 도구막대의 검색으로 지금 열려 있는 문서를 검색하고, 결과 위치로 바로 이동할 수 있습니다. 긴 PDF나 책에서 특정 단어를 찾을 때 유용합니다.

## 보관함 본문 검색

하단 `검색` 탭에서는 색인된 문서의 본문 전체를 검색합니다.

- 색인 대상: PDF, TXT, Markdown, EPUB, HWP/HWPX
- CBZ/CB7은 텍스트가 없는 문서로 처리됩니다.
- 결과는 파일명, 파일 타입, 위치(페이지/오프셋), 스니펫(본문 일부), 매칭 수와 함께 표시됩니다.

## 한글 검색 정책

한글은 검색어 길이에 따라 다른 방식이 쓰입니다.

- 3글자 이상: 부분 검색(가운데 글자 매칭에 강함)
- 1~2글자: 보조 검색으로 함께 찾아 줌

영문·숫자는 단어 단위로 검색되고, 파일명은 별도 색인으로 빠르게 찾습니다.

## 필터와 검색 결과 큐

검색 범위(scope)와 파일 타입으로 결과를 좁힐 수 있습니다. 결과 목록은 그대로 전환 큐가 되어, 한 문서를 연 뒤 두 손가락 스와이프로 다음·이전 결과 문서를 이어 볼 수 있습니다. 검색 → 비교 → 다음 문서로 이어지는 흐름이 끊기지 않습니다.

## 색인과 실패 처리

검색을 하려면 문서가 색인되어 있어야 합니다.

- 색인이 아직인 문서는 대기 상태로 표시됩니다.
- 색인에 실패한 문서는 `검색` 화면에서 다시 시도할 수 있습니다.
- 한 문서가 실패해도 다른 문서 검색에는 영향이 없습니다.

> 폴더를 통째로 색인하려면 `매직보관함`을 참고하세요.
