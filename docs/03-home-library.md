# 홈과 책장 보기

*책장 보기·정렬 메뉴*

홈은 `독서중` 영역과 책장 목록으로 구성됩니다. 위쪽 탭으로 책장을 바꾸고, 아래쪽 메뉴로 보기 방식을 조절합니다.

## 독서중

최근에 실제로 읽은 문서를 빠르게 이어 열기 위한 공간입니다. 읽고 있는 책이 없으면 비어 있는 안내가 표시됩니다. 여기에 보이는 순서는 "읽은 활동" 기준이라, 새로 읽을수록 위로 올라옵니다.

## 책장 종류

- `Ebook 책장`: 직접 추가한 문서. 추가한 순서를 유지합니다.
- `매직 책장`: 선택한 폴더 스캔으로 찾은 문서.
- 수동 책장과 포토함으로 문서를 직접 분류할 수도 있습니다.

## 보기와 정렬 메뉴

각 책장의 `{...}` 메뉴에서 다음을 바꿀 수 있습니다.

- 보기: 격자 / 목록
- 정렬: 최근순 / 제목순 / 확장자순 / 수동순
- 선택 모드: 여러 문서를 한 번에 관리

모든 책장은 타입과 문서 수에 관계없이 같은 메뉴 구성을 제공하므로, 한 곳에서 익히면 다른 책장에서도 그대로 쓸 수 있습니다.

## 표지에서 보이는 정보

격자 표지에는 썸네일, 파일 타입, 제목, 진행률 바가 표시됩니다. 진행률로 어디까지 읽었는지 한눈에 알 수 있고, 다 읽은 문서는 색과 짧은 상태로 구분됩니다.

확장자순으로 정렬하면 같은 포맷끼리 묶여, PDF·EPUB·만화 등을 그룹으로 훑어보기 좋습니다.

## 문서별 메뉴

표지의 보조 메뉴에서 문서 단위 작업을 할 수 있습니다.

- 문서 정보 보기
- 읽음 / 안 읽음 표시
- 텍스트 추출
- 책장에서 제거
- 삭제(확인 후 진행, 되돌릴 수 없는 작업은 빨간색으로 구분)

> 가이드 책장의 문서는 삭제·선택 삭제가 막혀 있습니다. 앱이 관리하는 문서이기 때문입니다.
